## Description

Proyecto API para un sistema de control de clases (Horario, asistencia, notas, etc)

## Version Node
```bash
$ node v18.18.0
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker postgres database
$ docker-compose up -d

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
